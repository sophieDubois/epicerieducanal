# Projet Epicerie du Canal

### Cloner le projet 
...

`git clone https://github.com/sophieDubois/epicerieducanal.git`
...

### Déplacer le terminal dans le dossier cloné
...
`cd epicerieducanal`
...

### Installer les vendors (pour recréer le dossier vendor)
...
`composer install`
...

### Lancer le serveur
...
`symfony serve`
...
